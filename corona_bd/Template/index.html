Corona Updates for BD API

This is a free API for Corona Updates in Bangladesh. Scraping data from Corona.gov.bd. Using free Heroku Account So This API will take a nap after 30 minutes of inactiveness. So that if the server is sleeping it might need some extra time for the first request to active the server then handel your request.
